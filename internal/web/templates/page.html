Quick start

使い方ガイド

左ペインで操作すると、実際に飛んだ HTTP リクエストとサーバーの生レスポンスが 右ペインのインスペクターに表示されます。1 つのレスポンスで 左の行と右のインスペクターが同時に書き換わるのが HTMX の OOB swap です。 右ペインを見ながら、次の順に試してみてください。

  1. POSTタスクを追加 → 新しい行が増え、インスペクターに 201 Created と HX-Trigger（トースト）が出る。
  2. PATCH✓ で完了トグル → 1 フィールドだけの部分更新。
  3. PUTタスク名を編集して Enter → title+done をまとめて差し替える全置換（冪等）。
  4. DELETE削除 → 確認ダイアログ（hx-confirm）の後、行が消える。
  5. 409同じ名前を再追加 → 重複は 409 Conflict。エラー応答も可視化される。
  6. GET検索ボックスに入力 → 300ms 後に絞り込み（hx-trigger + クエリ文字列）。All / Active / Done でフィルタ。
  7. ↻右上のテーマ切替 → Cookie 保存 + HX-Redirect でリロード。
